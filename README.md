# Sentiment Analysis on Social Media Apps

## Overview
This project performs sentiment analysis on social media text data and classifies user opinions as:

- Positive
- Negative
- Neutral

## Technologies Used
- Python
- TextBlob
- VADER Sentiment Analyzer
- Pandas
- Matplotlib
- Seaborn

## Features
- Text preprocessing
- Sentiment classification
- Visualization of sentiment distribution
- Comparison between TextBlob and VADER results

## Project Structure

data/ -> Dataset

src/ -> Python source code

outputs/ -> Generated charts

docs/ -> Project report

## How to Run

```bash
pip install -r requirements.txt
python src/sentiment_analysis.py
